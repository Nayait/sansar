Om Git og GitHub

Git, en type programvare, er et versjonskontrollsystem. Git distributerer koder. Det betyr at hver eneste bruker har en lokal kopi av kodebasen på det felles repositoryet, hvor koden i effekt sendes mellom forskjellige brukere. For å kunne kommunisere med et Git repository trengs en Git klient. Klienten består enten av kommandolinje eller et brukergrensesnitt. Git brukes til å utføre kommandoer for å sjekke endringer i filer, legge de til i et repository og synkronisere med felles repository.Git-klienten installeres lokalt på en maskin. Ved bruk av Github er det mulig å kommunisere med et repository gjennom nettleser.
GitHub er en tjeneste som tilbyr Git versjonskontroll. Rett og slett er GetHub menes å få. Dette er wevside som tillater opplasting repository på nett. Den gir sikkerhetskopiere til filer, gir visuelt grensesnitt og gir andre brukere mulighet til å respondere.
